Background

日本人の英語能力

国際ビジネスコミュニケーション協会(IIBC)が2020年に行ったアンケートによると 社会人が英語4技能の中で一番伸ばしたいと思っているものは スピーキング(話す能力)、次点でリスニング(聞く能力)でした。

シャドーイングの効果

シャドーイングを行うと英語の抑揚やリズム，強弱を鍛えることができます。

・スピーキング能力，リスニング能力の向上

・リーディングの速度向上

などの効果があるとされています。

モチベーションの維持

しかし、英語学習者でシャドーイングを続けている人はごく少数です。なぜでしょうか。

・既存の音源だと練習時に音声の再生・ストップが手間

・オリジナルのテキストでシャドーイングできない

以上のような原因から、私たちはシャドーイングのやる気をなくしてしまうと考えました。

Assistalkの開発へ

せっかく英語を勉強する気になったのに、そのような理由でやる気を失ってしまうのは非常にもったいない。

そのような想いから、私たちはシャドーイング支援アプリ「Assistalk」を開発しました。

How to use

1. お好きな文章を貼り付ける

2. Let's シャドーイングボタンを押す

たったこれだけ！

使い方はあなた次第！様々なことに利用できます

  • 研究論文

    詳しくみる

    Assistalkを用いた論文の翻訳

    Assistalkを使えば、論文を翻訳しシャドーイングすることが出来ます

    専門的な単語や文法を知ることもでき、一石二鳥！

    英語論文発表の発音の練習にも使えます

    • 現在大学生や大学院生で英語の論文を呼んでいる方

    • 社会人で輪読する機会がある方

    • 専門的な文章を読みたい方

    におススメです。

  • NEWSの記事

    詳しくみる

    NEWSの記事はAssistalkにおまかせ

    Assistalkを使えば、NEWSの記事も英語の教材として使えます。

    英語の勉強とともに、様々な分野の内容も知ることが可能です。

    • 英語の勉強とともに世界の動きを把握したい方

    • 現在大学生で、まだ本格的な研究をしていない方

    におススメです。

  • 著名人の演説

    詳しくみる

    演説の翻訳にもAssistalk

    皆さんは海外の著名人の演説の真似をしたことはありますか？

    Assistalkを使えば、長い文章でも自動で一文ずつ区切ってくれるので

    シャドーイングの練習に最適です。

    • 映画やドラマのセリフを真似して勉強したい方

    • 英語を勉強したいがとっかかりがない方

    におススメです。
